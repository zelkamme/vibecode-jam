# Моя Часть Работы: Frontend + Backend Orchestrator (N1)

Это — твой командный пункт. Здесь находится всё, за что отвечаешь ты: пользовательский интерфейс, с которым взаимодействует кандидат, и главный бэкенд-сервер, который управляет всей системой.

**Моя главная задача:** быть "оркестратором". Я не выполняю тяжелую работу (запуск кода, AI-анализ), а **делегирую** ее специализированным сервисам от N2 и N3.

## Запуск Моей Части

**Важно:** Для полноценной работы должны быть запущены оба моих сервиса. Откройте два терминала.

### 1. Запуск Frontend

```bash
# Перейдите в папку фронтенда
cd frontend

# Установите зависимости (если еще не сделали)
npm install

# Запустите сервер разработки
npm run dev
```
**Результат:** Пользовательский интерфейс будет доступен в браузере по адресу `http://localhost:5173`.

### 2. Запуск Backend (Оркестратор)

```bash
# Перейдите в папку бэкенда
cd backend

# Активируйте виртуальное окружение
# Windows:
.\\venv\\Scripts\\Activate.ps1
# macOS/Linux:
source venv/bin/activate

# Установите зависимости (если еще не сделали)
pip install -r requirements.txt

# Запустите FastAPI сервер
uvicorn main:app --reload
```
**Результат:** Мой главный API-сервер будет принимать запросы от фронтенда на `http://localhost:8000`.

## Эндпоинты Моего Оркестратора

Мой `main.py` предоставляет API для фронтенда.

*   `POST /api/run-code`
    *   **Что делает:** Принимает код от фронтенда и **ПЕРЕНАПРАВЛЯЕТ** запрос на Docker-сервис (N3) по адресу `http://localhost:3000/api/run-code`. Получает результат и возвращает его фронтенду.

*   `POST /api/chat`
    *   **Что делает:** Принимает историю чата от фронтенда и **ПЕРЕНАПРАВЛЯЕТ** запрос на ML-сервис (N2) по адресу `http://localhost:4000/api/interview`. Получает ответ от AI и возвращает его фронтенду.

*   `POST /api/analyze-integrity`
    *   **Что делает:** Принимает телеметрию с фронтенда (альт-табы, копипаста), считает базовый скор и вызывает ML-сервис (N2) для анализа стиля кода. Формирует и возвращает финальный Integrity Score.
